# 📱 Flutter Routing & Navigasi App  
**Project:** Routing-dan-Navigasi_p5_2306003  

## 📖 Deskripsi  
Aplikasi Flutter sederhana untuk menampilkan informasi produk dan navigasi antar halaman menggunakan **Named Routes**.  
Terdiri dari halaman beranda dan halaman detail.

---

## ✨ Fitur  
- Menampilkan gambar, nama, dan harga produk  
- Navigasi ke halaman detail  
- Menggunakan Named Routes  
- Struktur kode terpisah per halaman
